HelloWorld 的编写

#HelloWorld

    1.随便新建一个文件夹，存放代码
    2.新建一个java文件
        *文件后缀为.java
        *Hello.java
        *[注意点]系统可能没有显示文件夹后缀名，我们需要手动打开
    3.编写代码
        public class Hello{ //类 
            public static void main(String[] args){ //main方法
                System.out.print("hello world");
    

        }

        }
    4.编写javac java文件，会生成一个class文件
    5.运行class文件，java class文件

#可能出现的问题

    1.每个单词的大小不能出现问题，java是大小写敏感的
    2.尽量用英文
    3.文件名 和 类名必须保持一致，并且首字母大写
    4.符号使用了中文
